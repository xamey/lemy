## Dependency Licenses

Lemy depends on third-party packages that keep their own licenses. A local
dependency metadata scan found the Node dependency tree to be primarily MIT,
ISC, Apache-2.0, BSD, MPL-2.0, Unlicense, BlueOak, CC0, and CC-BY licensed,
with some packages not declaring license metadata in `package.json`.

One transitive build dependency currently declares `LGPL-3.0-or-later`
(`@img/sharp-libvips-darwin-arm64`, via the Vite/sharp toolchain). It is not
Lemy project code and remains governed by its own license. Python dependency
licenses are likewise governed by their upstream package metadata.

Apache-2.0 applies to Lemy source code and documentation in this repository,
not to third-party dependencies.
